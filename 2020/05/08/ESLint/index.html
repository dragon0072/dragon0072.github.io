2. ESLint 规则和配置

  1. rules 配置

    rules是配置ESLint的验证规则

    • 以字符串数组的形式，配置各种规则
      • 下标为 0 的字符串，配置当前规则是否启用
      • 下标为 1 的字符串，配置当前规则的逻辑
    • 每个规则，必须配置为下列三个选项中的一个
      • off或0 - 关闭规则
      • warn 或 1 - 开启规则，使用警告级别的错误（不会导致程序退出）
      • error 或 2 - 开启规则，使用错误级别的错误（当被触发时，程序会退出）

  2. extends

    一个配置文件可以被基础配置中的已启用的规则继承。

    • 例如 如果要使用 vue 的默认规则
